Do package.json pridaj alebo uprav scripts takto:

{
  "scripts": {
    "refresh-cache": "node scripts/refresh-cache-with-repair.js",
    "refresh-cache-original": "node scripts/refresh-cache.js"
  }
}

Ak nechceš prepísať pôvodný refresh-cache script, použi radšej:

{
  "scripts": {
    "refresh-cache-with-repair": "node scripts/refresh-cache-with-repair.js"
  }
}

A vo workflow potom daj:

npm run refresh-cache-with-repair

Voliteľné premenné:

REPAIR_LIMIT=300
REPAIR_TMDB_LIMIT=300
REPAIR_STREAM_LIMIT=300
REPAIR_TMDB=0       # vypne TMDB repair
REPAIR_STREAMS=0    # vypne stream/detailUrl repair
CACHE_FILE=data/cache.json
